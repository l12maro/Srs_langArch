There are different functionalities within the Tsuut'ina Language Archive. There are two main ways to interact with the materials: searching and browsing

Search

Users can search for words or sentences that appear within texts using the search function. To use this tool:

  • Click on "Search" on the navigation bar.
  • Insert the desired word or term on the search button.
  • (Optional) Specify further search requirements, such as the language, or the collection where you want to find the results

The search function looks for close or exact matches to the word that the user is searching for. The elements that are being searched are:

  • Metadata: That is, the terms that are used to classify a given element, like date of recording, genre, participants, speakers, etc. Currently we can look for all texts classified under a given language, genre or collection, or with certain speakers,
  • Title: Namely, the title of collections or sessions within which the different documents are classified.
  • Content: Currently, we can search for results present in the videos or audios that have been transcribed using ELAN software. There is no content search for other documents that contain no transcription or in other formats, such as pdfs.

Browse

If we are looking for a specific document, we can use the browse section to search for it. The browse section allows us to see the structure within which a document is classified. For example, we can access all documents classified within a given collection, a given session or a given speaker.

Other functionalities

Generate QR code:

To generate a QR code of a given page, right click and select "Create QR code for this page"

Download documents

You can download all documents in a given session by entering one of the documents and selecting the desired document to download from the sidebar menu.
